### 🚀 Featured projects

**[forenix-oss](https://github.com/thunderstornX/forenix-oss)** — _OSINT × Forensics, one workflow_
Open-source platform that turns public-source intelligence into **court-admissible evidence**: a SHA-256 forward-chained audit log, three independent attestation backends (local · GitHub · Sigstore Rekor), and a Structured-Analytic-Technique reasoning engine over six LLM providers.
`TypeScript` · `Next.js` · `Prisma` · live demo → **[forenix.tech](https://forenix.tech)**

**[lattice](https://github.com/thunderstornX/lattice)** — _Accountability layer for multi-agent AI_
Content-addressed, cryptographically-signed claim DAGs — tamper-evident provenance and a revocation waterfall for autonomous multi-agent systems.
`Python`

**[firmware-analysis-walkthrough](https://github.com/thunderstornX/firmware-analysis-walkthrough)** — _Reproducible firmware-security pipeline_
End-to-end, reproducible firmware analysis for OpenWRT 22.03 on ath79/MIPS.
`Reverse Engineering` · `Embedded`
